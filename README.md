# Árbol AVL – Segundo Estudio de Caso

Este proyecto implementa un Árbol AVL completamente funcional en Java.

Incluye inserción, eliminación, balanceo automático, cálculo de alturas, rotaciones LL, RR, LR y RL, e impresión visual del árbol en formato ASCII.

---

Funcionalidades principales

- ✔ Inserción de nodos con rebalanceo automático  
- ✔ Eliminación de nodos con rebalanceo  
- ✔ Cálculo de alturas  
- ✔ Factor de equilibrio  
- ✔ Rotaciones: **LL, RR, LR, RL**  
- ✔ Visualización del árbol en consola  
- ✔ Menú interactivo en consola (Java)
